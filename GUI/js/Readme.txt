**2. 설명 Instruction**	────────────────────────────────────────────────────────


			MeaXure Tamperer는 Sketch MeaXure 플러그인으로 출력한

			HTML 형식의 디자인 가이드 검수 과정에서 발생하는 불편함을 개선하고 

			업무 효율 증진에 도움을 줄 수 있는 다양한 인터페이스와 기능을 추가하는 도구입니다.


**3. 사용법 How to use**	────────────────────────────────────────────────────────



			1.	페이지 이동 단축키	단축키와 header 우측에 생성된 버튼(<, >)을 이용해 페이지 앞, 뒤를 이동합니다.

							● 앞(<) 페이지 이동 단축키		A / W / ← / ↑ / PgUp / 마우스휠 위로

							● 뒤(>) 페이지 이동 단축키		D / S / → / ↓ / PgDn / 마우스휠 아래로


			2.	양끝 페이지 이동 단축키	단축키를 이용해 가장 첫 페이지와 가장 마지막 페이지로 이동합니다.

							● 가장 앞 페이지 이동 단축키	Home

							● 가장 뒤 페이지 이동 단축키	End


			3.	사이드바 탭 전환 단축키	단축키를 이용해 좌측 사이드 바의 탭을 전환합니다.

							● artboards			Q

							● slices				E

							● colors				R


			4.	현재 페이지 시야 고정	좌측 사이드바 artboards 탭의 페이지 목록에서

							현재 보고있는 페이지가 항상 시야에 보이도록 고정합니다.


			5.	확대 축소 단축키		단축키를 이용해 viewer를 확대, 축소합니다.

							● 확대				Z / + / =

							● 축소				C / -


			6.	섹션 내비게이션		sketch 작업 파일에서 아래 조건에 따라 섹션을 구성하면

							viewer 좌하단 영역에 섹션 내비게이션이 자동으로 생성됩니다.

							섹션 내비게이션은 단축키 X키로 토글할 수 있습니다.

							ⅰ.	sketch 작업 파일에서 artboard 이름에 'cover', 'index', 'history'가 있을 때

								해당 페이지가 list item으로 추가됩니다.

							ⅱ.	이 중 history는 반드시 두 번째 item으로 삽입되므로,

								페이지 구성도 동일하게 관리하시기 바랍니다.

							ⅲ.	history가 여러 페이지일 경우

								가장 마지막(최신) 페이지만 list item으로 추가됩니다.

							ⅳ.	artboard 이름이 숫자로 시작하면 그 list item은 일정 너비 들여쓰기 됩니다.

								이를 이용해 가독성 높은 섹션 내비게이션을 구성할 수 있습니다.


			7.	현재 페이지 번호 표시	viewer 좌상단 영역에 현재 페이지 번호를 표시합니다.


			8.	활성화 객체 수 표시	현재 페이지 번호 우측에 현재 페이지에서 활성화되어있는 객체의 수를 표시합니다.

							Layer를 잠궈 관리해야하는 페이지의 잠금 누락과 더불어

							Sketch, MeaXure 플러그인의 시스템적 오류를 확인하는 지표로 사용합니다.


			9.	페이지 뛰어넘기		숫자(0~999) + enter 키를 눌러 언제든 원하는 페이지로 이동할 수 있습니다.

							ⅰ.	전체 페이지 수보다 큰 수를 입력하면 재입력을 요구하는 메시지가 잠시 나타납니다.

							ⅱ.	입력 중 ESC를 누르면 취소(초기화) 할 수 있습니다.

							ⅲ.	마지막으로 입력한 뒤 10초가 지나면 자동으로 취소(초기화)됩니다.
